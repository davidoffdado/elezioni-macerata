Dati e codice sorgente

Tutte le fonti usate in questo progetto sono pubbliche e liberamente scaricabili.

Fonti dei dati
  • Risultati elettorali per sezione Ministero dell'Interno tramite Eligendo
  • Viario elettorale (indirizzi seggi) Richiesta tramite FOIA all'Ufficio Elettorale del Comune di Macerata
  • Confine comunale Portale OnData sui confini amministrativi
  • Edifici e strade OpenStreetMap
  • Tile mappa Leaflet e CartoDB Positron
Scarica i dati
  • File Excel dei dati elettorali Dati utilizzati per l'analisi
  • File GeoJSON degli edifici con sezione assegnata edifici_colorabili_voti.geojson
  • File GeoJSON delle sezioni rappresentate dai poligoni di Voronoi voronoi_per_sezione.geojson
  • Codice sorgente completo github.com/davidoffdado/elezioni-macerata — link alla repository del progetto
